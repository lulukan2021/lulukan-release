# 小鹿快看

小鹿快看，一款精致的多平台桌面软件，用它管理绅士片库方便快捷。

主要功能：
1. 多目录映射为影片库管理
2. 多种分类方式方式：按演员、按系列、按分类
3. 支持多种搜索方式和排序方式
4. 支持多收藏夹设置
5. 支持设定评分和排序

计划中的功能：
1. 支持影片信息刮削
2. 内嵌播放器
